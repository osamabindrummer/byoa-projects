# Sitio estático con HTML, CSS, JavaScript y librerías Node

El plan de este proyecto del curso 'Build Your Own Apps' consiste en crear un sitio estático con HTML, CSS, JavaScript y librerías simples de Node. Lo que quiero es, primero, aterrizar un plan de trabajo con pasos claros sobre lo que Codex debe hacer y en qué debe ayudarme. A continuación hay una lista de tareas y actividades que abordaremos en esta sesión del curso:

## Plan

1. Crear una landing page simple.
2. Crear una plantilla para un post de blog.
3. Crear un conversor de Markdown a HTML.
4. Crear una integración simple para ConvertKit o Substack.
5. Añadir un formulario de contacto.
