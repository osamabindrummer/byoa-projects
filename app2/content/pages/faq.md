# Preguntas frecuentes

**¿Cómo está construido el sitio?**  
Con HTML, CSS y un pequeño script de Node que convierte Markdown en páginas estáticas.

**¿Cómo puedo suscribirme?**  
Completa el formulario de suscripción en la landing page y recibirás novedades por correo electrónico.

**¿Puedo contribuir contenido?**  
Claro, envíanos un mensaje a través del formulario de contacto y conversemos.
