# Build Your Own Apps — Registro de proyectos

Este repositorio recopila los proyectos que desarrollo como estudiante del curso **Build Your Own Apps** de Nat Eliason. La idea es centralizar cada ejercicio y mantener un historial claro de mi progreso a medida que avanzo en el programa.

## App 1 · Pomodoro Timer Minimalista - [Link](https://osamabindrummer.github.io/byoa-projects/app1/index.html)

- **Descripción:** Temporizador Pomodoro diseñado para priorizar el enfoque. Incluye modos de trabajo y descanso, un registro de objetivos por sesión y un contador de sesiones completadas.
- **Componentes principales:** Selector de modos (trabajo / pausas), temporizador dinámico con botones de iniciar/pausar, reinicio y extensión de +5 minutos, indicador visual con emojis, y registro de objetivos ingresados por cada pomodoro.
- **Uso:** [Abrir la app](https://osamabindrummer.github.io/byoa-projects/app1/index.html) directamente en el navegador para comenzar a trabajar con sesiones Pomodoro.
- **Estructura flexible:** El repositorio está preparado para alojar más aplicaciones del curso; cada nuevo proyecto tendrá su propia carpeta (`app2`, `app3`, etc.) con sus archivos dedicados y documentación asociada.

## App 2 · Sitio personal estático - [Link](https://osamabindrummer.github.io/byoa-projects/app2/dist/index.html)

- **Descripción:** Landing page minimalista con blog, sección “Acerca de”, preguntas frecuentes y contacto. Todo el contenido se escribe en Markdown y se genera como HTML estático mediante un script sencillo en Node.
- **Lo trabajado en la sesión:** Definí la estructura del proyecto (`content/`, `templates/`, `public/`), construí un generador estático con Node usando `marked`, añadí formularios (suscripción y contacto), diseñé un tema oscuro elegante en CSS y dejé instrucciones en `app2/README.md`.
- **Uso:** Cualquier persona puede abrir el enlace anterior y navegar el sitio directamente desde su navegador, sin instalaciones ni dependencias adicionales.

Más adelante se agregarán nuevas secciones para cada app que construya durante el curso, manteniendo esta guía como referencia única del progreso.
